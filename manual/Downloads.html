WebTest Downloads

This page contains links to the artifacts for the latest WebTest version, as well as for previous releases.

Latest Snapshot

Release class = development, build number = 4.0.0-RC2

This snapshot represents the current stream of development and can always be found here.

All releases contain the same set of files. The primary artifacts of interest are:

webtest.zip
Consult Building to build webtest from source.
Documentation
All documentation is served as GitHub pages. Includes the ant tasks manual, javadoc API, and self test results. They may be downloaded from the webtest gh-pages branch archive. If you only want the webtest manual, you can download it here.
Source code
Available in both zip and tar.gz formats.

Archived Canoo Downloads

The latest webtest downloads from Canoo can be found here. At the time of this writing, the latest source snapshot was tag R_1826, Dec 2014. The latest documentation is labeled 'development', Jul 2016.

If you are looking for something more stable and don't care about testing REST services, I used this release for several years before bothering to make my own fork.
