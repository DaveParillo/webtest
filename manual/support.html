    Online documentation

    The online documentation contains a lot of information about WebTest with numerous examples. The documentation of the steps is generated directly from the Java sources and therefore always up to date.

    Additionally you can look at WebTest's Self Test Report to see how steps are used.

    Mailing list

    WebTest's Mailing list is the right place for all questions concerning WebTest. Most of the time you will get a very helpful answer within a short time.
